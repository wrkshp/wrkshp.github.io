Get Started

What is Alexa?

Alexa is the assistant from Amazon. You can already do all sorts of things with Alexa, like play games, get information, order food, order items on Amazon, request a ride, control smart home, along with a multitude of other skills that developers have made to cater to this device.

Alexa has been out for three years, which has been plenty of time for developers to build their own apps. Now you can build one too using Alexa's Skills Kit.

Setup

Before coming into the workshop, you'll need...

  • An Amazon account
    Visit amazon.com and create an one, no credit card necessary.
  • An AWS account
    Please have this step done prior to the workshop.
    Visit aws.amazon.com to do create an account using the same email address. This one does require a credit card, but Amazon won't charge you. Learn more about Lambda. This also may require a 24-hour verification period.
  • Create an Amazon developer account
    Go to developer.amazon.com using the same email address.
  • A text editor
    We recommend Sublime or VIM, but any text editor works!
