A New Skill

Setup

Make sure you've made all the accounts necessary. If you haven't, go back to the Setup section and follow all of the steps.

Adding a New Skill

Navigate to developer.amazon.com. Click the Alexa tab, scroll to the bottom, and select the Alexa Skills Kit. Click Start a Skill in the top right corner. Sign into your Amazon account. If you haven't created a developer account yet, you'll be prompted to do so now.

Once you've reached the developer home page, select Alexa Skills Kit and hit Create Skill.

Now we need to fill in a few fields to identify our skill.

  • Skill Name is the name shown in the skills store. It can be anything you want.
  • Skill Type is our custom interaction model, which includes our intents and slots. It's our custom way of talking to Alexa. Select this type and hit Create Skill at the top to continue.

You'll be taken to the Skills Developer landing page for your skill. We'll go through the Skill Builder Checklist now. You can also see this checklist in the left menu.

We'll do the easiest step first: the Invocation Name. This is what name we use when we talk to Alexa. For example, if our invocation name is "echomon", we would say, "Ask echomon to ____".

Hit Save Model at the top, then select Intents in the left menu. Or, from the Skills Builder Checklist on the Landing Page, click on the Intents, Samples, and Slots item in the checklist to take you straight to the intent creation page.
