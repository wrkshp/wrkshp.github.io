Interaction Model

Click into the Interaction Model and launch the Skill Builder.

Creating an Intent

We'll create a custom intent called GetPokemonIntent. For this intent, we're going to add custom slots on the right hand side. We'll add one called PokedexNumber, which we're going to use to ask Alexa for a specific Pokemon. We'll choose a slot type of AMAZON.NUMBER, which is one of Amazon's few prebuilt types.

Adding Some Utterances

Now, we'll define some sample utterances. This is what we're going to ask Alexa to invoke the GetPokemonIntent. For example:

"What is the name of pokemon {PokedexNumber}"

You can go ahead and add as many of these as you want. Let's make a simpler one.

"Who is {PokedexNumber}"

Generally a production skill will have over 100 of these to cover all the phrasings that a user might use. The more you add, the better it is for your customers to use it.

Save Your Skill

To save and submit your skill, MAKE SURE YOU CLICK SAVE MODEL FIRST. If you don't, you may lose all your work!

After you save your model, click Build Model. This may take a few seconds.

Now we'll configure our skill.
