A New Skill

Setup

Make sure you've made all the accounts necessary. If you haven't, go back to the Setup section and follow all of the steps.

Adding a New Skill

Navigate to developer.amazon.com and sign in. Now, click the Alexa tab and select the Alexa Skills Kit. Click Add a New Skill in the top right corner.

Now we need to fill in a few fields to identify our skill.

  • Skill Type is our custom interaction model, which includes our intents and slots. It's our custom way of talking to Alexa.
  • Language. Choose English for now, or whatever language you want to interact with Alexa with.
  • Name is the name shown in the skills store. It can be anything you want.
  • Invocation Name is what name we use when we talk to Alexa. For example, if our invocation name is "Echomon", we would say, "Ask Echomon to ____".
  • Global Fields. We don't need to worry about this for now. This is mostly if you want your skill to play music. For our purposes, we will be saying no.

Hit Save to move onto the Interaction Model.
